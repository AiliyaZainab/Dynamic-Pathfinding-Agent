# Dynamic-Pathfinding-Agent
Dynamic Pathfinding Agent in Python using GBFS and A* algorithms. Handles dynamic obstacles, real-time re-planning, and GUI visualization of paths, visited nodes, and metrics.
# Dynamic Pathfinding Agent

## Requirements
- Python 3.x
- Tkinter (usually pre-installed)
- Optional: Pygame if using Pygame version

## Run
```bash
python main.py
